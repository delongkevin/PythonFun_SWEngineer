# Virtual ADAS ECU

A Python-based virtual **Advanced Driver Assistance System (ADAS) Electronic Control Unit** for demonstration and testing purposes.

![ADAS ECU Demo Dashboard](https://github.com/user-attachments/assets/2831219b-e068-49ee-a8d3-ad92007578ea)

---

## Features

| Feature | Description |
|---|---|
| **4 Virtual Cameras** | Front (120В° / 50 m), Rear (120В° / 30 m), Left (180В° / 10 m), Right (180В° / 10 m) |
| **12 USS Sensors** | 4 front bumper + 4 rear bumper + 2 front-flank + 2 rear-flank |
| **Autonomous Parking** | Parallel, perpendicular, and angled-bay modes with multi-arc path planning |
| **Trailer Hitch Assist** | Rear-camera + USS fusion for tow-ball alignment guidance |
| **Trailer Steering Guidance** | Articulation-angle tracking, anti-jackknife, predictive steering corrections |

## Quick Start

```bash
# Install dependencies
pip install -r ADAS_Virtual_ECU/requirements.txt

# Run the demo (all features, 30 frames)
python ADAS_Virtual_ECU/demo/adas_demo.py --mode all --steps 30 --output-dir /tmp/adas_out

# Run a single-feature scenario
python ADAS_Virtual_ECU/demo/adas_demo.py --mode parking
python ADAS_Virtual_ECU/demo/adas_demo.py --mode hitch
python ADAS_Virtual_ECU/demo/adas_demo.py --mode tsg

# Run unit tests
python -m unittest ADAS_Virtual_ECU/tests/test_adas_ecu.py -v
```

## ECU API Usage

```python
from ecu import ADASECUCore
from ecu.ecu_core import ECUInputs
from ecu.parking import ParkingType

ecu = ADASECUCore(seed=42)

# Activate features
ecu.activate_parking(ParkingType.PERPENDICULAR)
ecu.activate_tha()
ecu.activate_tsg()

# Step the ECU (call at 10вҖ“100 ms intervals)
inputs = ECUInputs(
    scene_objects=[
        {"type": "parking_slot", "distance_m": 3.0, "azimuth_deg": 0.0},
    ],
    uss_distances={"USS_R1": 1.2, "USS_R2": 1.0},
    vehicle_speed_mps=-0.8,
    steering_angle_deg=-20.0,
    trailer_hitch_offset_px=(10, 5),
    trailer_articulation_deg=15.0,
)

outputs = ecu.step(inputs)

print(outputs.system_health)                          # "OK" or "CRITICAL OBSTACLE вҖ” ..."
print(outputs.parking_status.guidance_text)           # Human-readable parking guidance
print(outputs.tha_status.steering_correction_deg)     # ВұВ° steering correction
print(outputs.tsg_status.jackknife_risk)              # 0.0вҖ“1.0 jackknife risk score
```
