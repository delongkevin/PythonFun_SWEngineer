Steps:
1. Run install_dependencies.bat.  Make sure all dependencies are installed, if not, reinstall python 3.12 or newer.
2. Left Click on RUN_APP.bat to run application.
3. Application will load - make sure you upload dbc and cdd files that corresponds to he can log first.
4. Then Upload the log file to app
5. You an start replay - if connected to ECU, it will try to connect to replay tx messages onto CVADAS, otherwise, it will create a virtual environment for replay if not able to connect.
4. report will be generated after replay of can log was completed.
